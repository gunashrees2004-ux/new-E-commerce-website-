LiveDemo:

## Description
This project is a ecommerece project where user can navigate through different routes, and browse the products, then the user can also add products to cart list

## Feature
-user authentication
-product listing based on user login details
-state management using state of Component package from re rendering the Components based on state updates
-component based archicheture for reuseability of code or component
-navigating between different routes using React-Router
-For implementing a cart list I have used react context concept
-responsive website has been implemented using media queries and flex box layouts
